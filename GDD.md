# Nova Pop — One Page GDD

**장르**: 버블 슈터 | **테마**: 우주 (Space) | **플랫폼**: iOS / Android

---

## 핵심 루프
대포로 버블을 발사 → 같은 색 3개 이상 연결 → 제거 → 반복

## 차별화 메카닉

**특수 버블**
| 버블 | 효과 |
|------|------|
| 레인보우 | 모든 색 매칭 가능 |
| 블랙홀 | 기존 폭탄 버블 대체 — 터지면 주변 버블 흡수 제거 |
| 유성 | 직선 관통, 소행성도 제거 가능 |

**필드 요소**
| 요소 | 효과 |
|------|------|
| 소행성 | 새 줄 추가 시 랜덤 위치에 섞여 등장. 제거 불가 (유성 버블 제외), 버블이 부딪히면 튕겨남 |
| 웜홀 | 좌/우 벽에 랜덤 타이밍으로 생성, 몇 턴 후 사라짐. 들어간 버블이 반대편 다른 높이로 나옴 |

## 시스템
| 항목 | 내용 |
|------|------|
| 재화 | 골드 단일 |
| 경제 | 게임 종료 시 점수에 따라 배수로 골드 지급 |
| 상점 | 골드로 스킨 구매 |
| 부활 | 광고 시청 1회 |
| 스킨 | 버블 스킨 / 대포 스킨 예정 |
| 진행 방식 | 무한 모드 |

## 난이도 스케일링 (점수/시간 기반)
| 요소 | 초반 | 후반 |
|------|------|------|
| 소행성 출현 확률 | 낮음 | 높음 |
| 웜홀 등장 빈도 | 낮음 | 높음 |
| 새 줄 추가 속도 | 느림 | 빠름 |
| 특수 버블 출현 확률 | 낮음 | 높음 |

> 구체적 수치는 플레이테스트 후 조정

## 미정

- 블랙홀 흡수 범위
- 난이도 구간 기준 (점수 기반 vs 시간 기반)
